# Prompts Used

## Prompt 1: Understand the Assignment

I need to build a simple local prototype for a Product Management internship assignment. The tool should take an image as input, analyze pixels, map colors into 11 categories, and generate percentage charts. Help me design the product approach, algorithm, and deliverables.

## Prompt 2: Build the Prototype

Create a Python prototype that:
- accepts an image path
- extracts RGB pixels
- converts RGB to HSV
- maps every pixel to one of 11 color categories
- calculates percentage distribution
- generates a bar chart
- exports CSV results

Use Pillow, NumPy, Pandas, and Matplotlib.

## Prompt 3: Improve the Product Thinking

Make the project look recruiter-ready. Add:
- PRD structure
- user personas
- edge cases
- success metrics
- tradeoffs
- future roadmap
- developer handoff notes

## Prompt 4: Interview Explanation

Write a short explainer script where I explain the product requirement, why HSV was used, how the algorithm works, and how the output helps users.
